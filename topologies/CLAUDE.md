# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is the `topologies/` subfolder of **oakphylo2026**, an oak phylogenetics analysis project for a book chapter. It compares phylogenetic trees (topologies) generated from three genomic datasets for *Quercus* (oak) species.

## Running the Analysis

**Full pipeline:**
```r
source("scripts/000_doItAll.R")
```

**Individual scripts** ‚Äî run in numbered order from an R session with working directory set to the repo root:
```r
source("scripts/00_readData.R")    # load and preprocess trees
source("scripts/10_compareTrees.R") # prune to focal clades
source("scripts/20_ordination.R")  # MDS on tree distances
# etc.
```

**Key flags at the top of `scripts/000_doItAll.R`:**
- `globalDoPDF = FALSE` ‚Äî set `TRUE` to regenerate all PDF outputs
- `rogues = FALSE` ‚Äî set `TRUE` to rerun rogue taxa analysis (slow)

## Architecture

### Three Datasets
All analyses operate on three named datasets accessed via `trees[[dataset]][[analysis_type]]`:
- `empiricalRAD` ‚Äî empirical RAD-seq reference-guided assembly (RAxML trees)
- `simRAD` ‚Äî simulated RAD-seq data
- `reSeq` ‚Äî pseudoreference genome sequences

### Pipeline Stages (script order)
| Script | Role |
|--------|------|
| `00_readData.R` | Load `.tre` files, label tips with taxonomy, root trees |
| `05_ipyradStats.R` | Summarize ipyrad assembly statistics |
| `07_raxmlStats.R` | Extract RAxML run summaries |
| `10_compareTrees.R` | Prune trees to 18 focal clades for cross-dataset comparison |
| `20_ordination.R` | MDS on Robinson-Foulds / clustering-information distances |
| `25_treeComparisons.R` | Cophyloplots, consensus trees, tree islands |
| `27_handyTreeStats.R` | Tree statistics |
| `30_cladeLoadings.R` | Clade support across analyses |
| `40_rogues.R` | Rogue taxa identification (Rogue package) |

### Data / Output Layout
- `data/` ‚Äî input tree files (`.tre`), Excel metadata, ipyrad stats; **not generated by these scripts**
- `out/figures/`, `out/tables/`, `out/ordinations/`, `out/trees/`, `out/rogues/`, `out/SUPPLEMENT/` ‚Äî generated outputs

### Key Conventions
- Tip labels format: `Quercus species|specimen_code`; stripped to species only for cross-dataset comparisons
- Trees rooted on *Lithocarpus* (or *Castanopsis* for empirical data)
- Bootstrap replicate trees named with `.bt###` suffix and kept separate from ML trees
- `multiPhylo` objects used throughout; metadata in list-based dataframes with row names matching tip labels
