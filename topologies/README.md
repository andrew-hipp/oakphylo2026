# topologies/

R pipeline for comparing phylogenetic tree topologies across three genomic datasets for *Quercus* (oak) species. The pipeline loads RAxML trees, prunes them to focal clades, runs MDS ordination on tree distances, builds cophyloplots and consensus trees, summarizes clade support, and identifies rogue taxa.

Scripts by Andrew Hipp (ahipp@mortonarb.org).

---

## Datasets

All analyses operate on three datasets, accessed throughout as `trees[[dataset]][[analysis_type]]`:

| Dataset | Description |
|---------|-------------|
| `empiricalRAD` | Empirical RAD-seq data assembled against reference genomes (RAxML) |
| `simRAD` | RAD-seq data simulated with RADinitio from whole-genome sequences (RAxML) |
| `reSeq` | Pseudoreference whole-genome resequencing (RAxML) |

A reference tree from Hipp et al. (2020) is also loaded as `tr2020` for comparison.

## Running the Pipeline

**Full pipeline** — from an R session with working directory set to `topologies/`:

```r
source("scripts/000_doItAll.R")
```

**Individual scripts** — source in numbered order:

```r
source("scripts/00_readData.R")
source("scripts/05_ipyradStats.R")
source("scripts/07_raxmlStats.R")
source("scripts/09_sitePatterns.R")
source("scripts/10_compareTrees.R")
source("scripts/20_ordination.R")
source("scripts/25_treeComparisons.R")
source("scripts/27_handyTreeStats.R")
source("scripts/30_cladeLoadings.R")
source("scripts/40_rogues.R")
```

**Key flags** (set at the top of `scripts/000_doItAll.R`):

| Flag | Default | Effect |
|------|---------|--------|
| `globalDoPDF` | `FALSE` | Set `TRUE` to regenerate all PDF figures |
| `rogues` | `FALSE` | Set `TRUE` to rerun rogue taxa analysis (slow) |

## Scripts

| Script | Role |
|--------|------|
| `000_doItAll.R` | Master script; sources all others in order |
| `00_readData.R` | Load `.tre` files and bootstrap replicates, label tips with taxonomy, root trees; produces `trees`, `boots`, `tr2020` |
| `05_ipyradStats.R` | Parse ipyrad assembly statistics; identify coverage and locus outliers; produces `dat_ipy`, `dat_indVects`, `dat_hilo` |
| `07_raxmlStats.R` | Extract base-pair counts and RAxML run metadata; produces `dat_bp`, `dat_rax` |
| `09_sitePatterns.R` | Parse `Alignment Patterns` from all RAxML info files; summarize mean and SD per dataset (`simRAD`, `reSeq`, `empiricalRAD`, `empiricalRAD_denovo`); writes `out/tables/raxml_site_patterns.csv` |
| `10_compareTrees.R` | Prune trees to 18 focal clades with standardized tip labels for cross-dataset comparison; produces `treesAll.clades` |
| `20_ordination.R` | MDS on Robinson-Foulds / clustering-information distances; produces `trees.points` (2-D coordinates) and `trees.islands` |
| `25_treeComparisons.R` | Cophyloplots, consensus trees, tree-island summaries; produces `treesAll.cophylos`, `trees.con` |
| `27_handyTreeStats.R` | Monophyly statistics per clade, dataset, and reference genome; writes `out/ordinations/ordinationStats.csv` |
| `30_cladeLoadings.R` | Envfit of clade presence/absence onto ordination space (vegan); writes ordination PDFs to `out/ordinations/` |
| `40_rogues.R` | Rogue taxa identification (Rogue package); writes per-analysis CSVs to `out/rogues/` |
| `99_scratch.R` | Scratch / development code; not sourced by `000_doItAll.R` |

## Data Layout

Input data live in [`data/`](data/) and are **not generated by these scripts** — they come from upstream RAxML and ipyrad runs. See [data/README.md](data/README.md) for a full description of subdirectories and metadata files.

Key conventions:
- Tip labels format: `Quercus species|specimen_code`; stripped to species only for cross-dataset comparisons
- Trees rooted on *Lithocarpus* (or *Castanopsis* for empirical data)
- Bootstrap replicate trees named with `.bt###` suffix and kept separate from ML trees

---

## Output Layout

All outputs are written to [`out/`](out/) and are regenerated by sourcing `000_doItAll.R`. See [out/README.md](out/README.md) for details.

| Directory | Contents |
|-----------|----------|
| `out/figures/` | Main manuscript figures (`FIG2_*` – `FIG7_*`) as PDFs |
| `out/SUPPLEMENT/` | Supplementary figure PDFs (`FIGS1*` – `FIGS3*`) |
| `out/ordinations/` | Per-clade and per-dataset MDS plots and envfit overlays |
| `out/tables/` | CSV summary tables (`monophylyMat.csv`, `ordinationStats.csv`, `raxml_site_patterns.csv`) |
| `out/trees/` | Pruned, consensus, and cophylo tree PDFs |
| `out/rogues/` | Rogue taxa results per analysis (`rogues_*.csv`; requires `rogues = TRUE`) |
