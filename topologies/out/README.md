# out/

All output generated by the analysis pipeline. Contents are recreated by sourcing `scripts/000_doItAll.R`; do not edit files here by hand.

## Subdirectories

| Directory | Contents | Generating script(s) |
|-----------|----------|----------------------|
| `figures/` | Main manuscript figures (`FIG2_*` through `FIG7_*`) as PDFs | `25_treeComparisons.R`, `20_ordination.R` |
| `SUPPLEMENT/` | Supplementary figure PDFs (`FIGS1*`–`FIGS3*`), one per analysis run | `25_treeComparisons.R`, `20_ordination.R` |
| `ordinations/` | MDS/ordination plots and envfit overlays for individual clades and datasets | `20_ordination.R` |
| `tables/` | CSV summary tables (`monophylyMat.csv`, `ordinationStats.csv`) | `30_cladeLoadings.R`, `20_ordination.R` |
| `trees/` | PDFs of pruned, consensus, and cophylo trees | `25_treeComparisons.R`, `10_compareTrees.R` |
| `rogues/` | Per-analysis rogue taxa results as CSVs (`rogues_*.csv`) | `40_rogues.R` (requires `rogues = TRUE` in `000_doItAll.R`) |
